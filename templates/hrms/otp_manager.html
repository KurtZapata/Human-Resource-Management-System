📋 How Auto-OTP Works

Admin Screen:
  1. A new OTP is generated automatically every 3 minutes.
  2. Display this page on a screen employees can see, or use Full Screen.
  3. Use Rotate Now to force an immediate new code.
  4. Switch to Employee-Specific to lock a code to one person.
Employee Steps:
  1. Log in at the attendance portal.
  2. Click Time In or Time Out.
  3. Type the 6-digit code shown on the screen.
  4. A confirmation shows their logged time.
⚠ Each OTP code can only be used once per employee within its validity window.
